### Graficas Usadas para el Analisis
<br>

### Grafica de Barras

<img src="https://github.com/gkruiz/SEMI2_Practica1_2024/blob/proyecto_2/IMAGENES/P2/barras.png?raw=true" width="800px">

este tipo de grafica se uso para presentar un grupo de datos que no pasa de los 500 items para presentar un listado de items y ademas de eso el poderle asignar lo que es un valor , con esto podemos visualizar de mejor manera el valor que le corresponde a cada item y asi mismo poder comparar su valor con los demas ,es mas util para presentar los datos de manera individual y su valor asociado de manera horizontal ,si son muchos y tambien para hacer su comparativa con el maximo y minimo unicamente corriendo la barra desplazadora

<br>

### Grafica de Areas

<img src="https://github.com/gkruiz/SEMI2_Practica1_2024/blob/proyecto_2/IMAGENES/P2/areas.png?raw=true" width="800px">


este tipo de grafica se uso para presentar lo que es una comparitva entre poblacion y numero de muertes acumuladas ,se uso principalmente porque presenta la poblacion de todos los municipios , ademas de eso , se observa de mejor manera lo que es la relacion entre la poblacion y el numero de casos acumulados por cada municipio, y se observa bien que no importa mucho la poblacion , pues los valores incluso en los municipios con menos poblacion tienen casos similares a los de mayor poblacion ,principalmente se uso para mostrar varios valores en una unica vista y que estos fueran digeribles , donde no es tan necesario el presentarlos de manera individual , sino mas bien en conjunto y verlos en una linea de crecimiento o decrecimiento en el eje x 

<br>

### Grafica de Anillos

<img src="https://github.com/gkruiz/SEMI2_Practica1_2024/blob/proyecto_2/IMAGENES/P2/anillos.png?raw=true" width="800px">


este tipo de grafica ,se uso para presentar los valores de una manera mas simple ,pues nos muestra un circulo que representa el total , y cada segmento es equivalente a un porcentaje del total, con este podemos visualizar bien que porcentaje del total representa cada elemento , este en general se uso mas para presentar datos de un valor y una etiqueta , ambos para mostrar el porcentaje de aporte frente al total de una forma simple 

<br>


### Grafica de Pastel

<img src="https://github.com/gkruiz/SEMI2_Practica1_2024/blob/proyecto_2/IMAGENES/P2/pastel.png?raw=true" width="800px">


este tipo de grafica ,se uso para presentar los valores de una manera mas simple ,pues nos muestra un circulo que representa el total , y cada segmento es equivalente a un porcentaje del total, con este podemos visualizar bien que porcentaje del total representa cada elemento ,viene a ser similar al de Anillos , para mostar uno menos denso se uso el de anillos y mas denso el de pastel ,este se uso principalmente para presentar los valores de un valor y una etiqueta de manera simple y el aporte que da cada uno al total de datos que seria la suma 

<br>


### Grafica de Embudo

<img src="https://github.com/gkruiz/SEMI2_Practica1_2024/blob/proyecto_2/IMAGENES/P2/embudo.png?raw=true" width="250px">


este tipo de grafica, se uso para presentar los datos y sus valores en una sola vista y ademas de eso podamos comparar un valor con el resto y ver tanto el valor maximo y el minimo de una sola vez ,es facil de digerir los datos pues no son muchos y ademas de eso se visualizan todos los datos y se presentan en orden , este sirvio principalmente para mostrar pocos valores y asi mismo presentarlos en una unica vista , y tambien tener el maximo y minimo en los extremos para realizar comparaciones 

<br>

### Grafica de Tree Map

<img src="https://github.com/gkruiz/SEMI2_Practica1_2024/blob/proyecto_2/IMAGENES/P2/treemap.png?raw=true" width="800px">


este tipo de grafica se usa para presentar en un rectangulo el total de los datos , ademas de eso este muestra el aporte al total de una forma muchisimo mas entendible para el usuario, donde el valor que esta mas hacia la izquierda y mas hacia arriba es el mas significavo mientras que los que estan mas a la derecha y hacia abajo son menos significativos , el total de las areas presentadas viene a ser la suma total de los datos ,se uso principalmente para tener pocos valores en una caja y ver con areas de manera mas sencilla el aporte individual al total sin tener que ver los numeros asignados
